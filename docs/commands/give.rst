!give
=====

Usage
    !give ``<amount>`` ``<username>`` ``[username]`` ``[username]`` ...

Example
    Give currency by username::

        !give 500 @username1 @username2

Usage
    !give ``<amount>`` all

Example
    Give currency to everyone::

        !give 500 all


Usage
    !give `<amount>` chat

Example
    Give currency to everyone in chat::

        !give 500 chat
