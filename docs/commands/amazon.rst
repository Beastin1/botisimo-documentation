!amazon
=======
Search Amazon products. Returns top 10 results in Discord and top result in Twitch, Mixer, & Youtube

Usage
-----
!amazon `<search>`

Example
-------
::

    !amazon dr. seuss

Searches amazon for "dr. seuss" and returns top result(s)
