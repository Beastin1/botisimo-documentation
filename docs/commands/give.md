# !give
Give currency to 1 or more users

#### Usage
!give `<amount>` `<username>` `[username]` `[username]` ...

#### Example
    !give 500 @username1 @username2

Give currency to everyone

#### Usage
!give `<amount>` all

#### Example
    !give 500 all

Give currency to everyone currently in chat

#### Usage
!give `<amount>` chat

#### Example
    !give 500 chat
