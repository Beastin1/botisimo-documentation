!vote
=====

The ``!vote`` command is used to vote in `polls <https://botisimo.com/account/polls>`_ via chat.

.. warning::

    Casting multiple votes for a poll may disqualify you from the poll if enabled by the streamer.

Usage:
    !vote ``<option>`` ``[bet]``
