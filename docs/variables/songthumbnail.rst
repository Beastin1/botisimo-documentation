$(songthumbnail)
================

Resolve the url to the thumbnail of the current song playing on the `music player <https://botisimo.com/account/music>`_.

Usage:
    $(songthumbnail)

Example Command:
    **name**: !songthumbnail

    **response**: $(songthumbnail)

    **output**::

        user:     !songthumbnail
        botisimo: https://i1.ytimg.com/vi/vBGiFtb8Rpw/mqdefault.jpg
