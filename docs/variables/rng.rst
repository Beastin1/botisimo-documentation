$(rng)
======

Resolve a random number between given numbers.

Usage:
    $(rng ``<min>`` ``<max>``)

Example Command:
    **name**: !roll

    **response**: You rolled a $(rng 1 100)

    **output**::

        user:     !roll
        botisimo: You rolled a 76
        user:     !roll
        botisimo: You rolled a 33
