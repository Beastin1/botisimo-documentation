$(rng)
======

Resolve a random number between given numbers.

Usage
    $(rng ``<min>`` ``<max>``)

Example
    Output a randomly selected number between 1 and 100 (including 1 and 100)::

        You rolled a $(rng 1 100)

    ``!roll`` -> ``You rolled a 76``

    ``!roll`` -> ``You rolled a 33``
