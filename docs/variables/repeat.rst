$(repeat)
=========

Resolves a repeated string.

Usage:
    $(repeat ``<count>`` ``<string>``)

Example Command:
    **name**: !repeat

    **response**: $(repeat 5 PogChamp Kappa)

    **output**::

        user:     !repeat
        botisimo: PogChamp Kappa PogChamp Kappa PogChamp Kappa PogChamp Kappa PogChamp Kappa
