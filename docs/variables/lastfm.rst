$(lastfm)
=========

Resolves the last scrobbled song from lastfm for the given username (must be lastfm username).

Usage:
    $(lastfm ``<username>``)

Arguments:
    * ``username`` **<required>** - The name of the lastfm user

Example Command:
    **name**: !lastfm

    **response**: $(lastfm botisimo)

    **output**::

        user:     !lastfm
        botisimo: Egzod - Universe
